Хоккей

Хоккейные коньки "Ледовый мастер"

  • Материал ботинка: высокопрочный нейлон с водоотталкивающим покрытием.
  • Лезвие: нержавеющая сталь с повышенной износостойкостью.
  • Фиксация: усовершенствованная система шнуровки с автоматическим затягиванием.

Хоккейная клюшка "Штормовой удар"

  • Материал: композит, сочетающий углеродное волокно и стекловолокно.
  • Длина: доступна в различных размерах.
  • Угол крюка: различные варианты для адаптации.

Беговые лыжи

Беговые лыжи "Северный след"

  • Конструкция: легкий деревянный сердечник с карбоновым армированием.
  • Покрытие: синтетическая смола для улучшения скольжения.
  • Крепления: совместимость с ведущими брендами.

Палки для беговых лыж "Экспресс-темп"

  • Материал: карбоновое волокно для легкости и прочности.
  • Рукоятки: эргономичные, комфортные.
  • Ремешки: легко регулируемые для быстрой настройки.

Сноубординг

Сноуборд "Горный вихрь"

  • Конструкция: гибридный профиль с кэмбером и рокером.
  • Материалы: деревянный сердечник с углеродным волокном.
  • Покрытие: синтетическая эластомерная пленка.

Крепления для сноуборда "Стихия"

  • Система фиксации: легкосъемные с микрорегулировкой.
  • Материал: высокопрочный алюминиевый сплав.
  • Подошва: подходит для всех типов сноубордических ботинок.
